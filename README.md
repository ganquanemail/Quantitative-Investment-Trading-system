## 二、“量化投资交易策略分析与系统设计”
### 0. 为实现\<trading system>中的量化交易系统,trading_system文件夹下，子文件夹内有相应README.md和数据文件；
### 1. 实现思路为：使用掘金客户端，批量生成不同参数的策略文件，代码中使用函数在回测结束时保存回测指标json用于筛选参数，具体的某一个回测详细结果json文件在客户端查看回测时手动下载用于分析最优结果详细数据；
